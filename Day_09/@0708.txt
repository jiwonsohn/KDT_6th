******함수 선언 후 호출해서 사용시,  함수 내 변수 처리 현황 확인 방법

1. 실행버튼 옆 옵션을 눌러 'python debugger: debug python file' 클릭

2. 호출한 함수가 있는 line을 클릭하면 빨간 점 생성 (break point 생성!)
		(break point=> 단계를 확인할 코드 Line)

3. 왼쪽 'Run and Debug' 클릭

4. Locals -> Function Variables-> 함수 목록 확인

5. 위에 아래방향 화살표 누르면서 변수가 함수에서 어떻게 처리되는지 확인

6. 끝나면 해당 코드 Line 눌러 break point 해제
